# flappy bird

- **Download release**, **extract**, open **cmd** (Windows) or **Terminal** (macOS/Linux) in the project folder and run

## Linux (Arch)

```sh
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Windows

```sh
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## macOS

```sh
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
